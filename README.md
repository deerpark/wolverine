# Wolverine
레진코믹스의 웹툰의 년도별 랭킹을 확인 할 수 있습니다.

## How to use

Install it and run:

```bash
npm install
npm run dev
# or
yarn
yarn dev
```

Deploy it to the cloud with [now](https://zeit.co/now) ([download](https://zeit.co/download))

```bash
now
```
